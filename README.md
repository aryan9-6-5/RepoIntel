## Description

GitHub Profile Explorer is a modern web application that allows users to explore public GitHub profiles and repositories with AI-powered insights. Search any GitHub user, view their repositories with advanced filtering options, and get intelligent summaries generated by LLM technology. The application uses a secure architecture where all AI API keys are stored server-side in Supabase Edge Functions, ensuring no sensitive data is exposed to the client.

## Features

-  Search any public GitHub profile by username
-  Browse all public repositories with detailed metadata
-  AI-generated summaries for repositories and developer profiles
-  Advanced filtering by stars, language, and update date
-  Aggregate statistics (total repos, stars, language distribution)
-  Sort by popularity, recency, or activity
-  Secure backend with no API keys exposed to client
-  Fully responsive design

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/aryanhanumakonda/github-profile-explorer.git
   cd github-profile-explorer
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env.local` file in the project root (see Configuration section)

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open `http://localhost:5173` in your browser

## Configuration

Create a `.env.local` file in the project root with the following variables:

```env
# Supabase Configuration
VITE_SUPABASE_URL=https://your-project-id.supabase.co
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

# Optional: GitHub Personal Access Token (for higher rate limits)
VITE_GITHUB_TOKEN=your_github_personal_access_token
```

## Usage

### Running the Application

To start the development server:
```bash
npm run dev
```

To build for production:
```bash
npm run build
npm run preview
```

### Using the Application

1. Enter a GitHub username in the search bar
2. Click "Search" or press Enter
3. View the user's profile information and repositories
4. Use filters to narrow down repositories by:
   - Programming language
   - Star count
   - Last update date
5. Click "Generate AI Summary" on any repository or profile for intelligent insights
6. Sort repositories by stars, forks, or update date

### Example API Usage

The application communicates with Supabase Edge Functions for AI summaries:

```typescript
const { data, error } = await supabase.functions.invoke('summarize-repo', {
  body: {
    type: 'repo', // or 'profile'
    data: {
      name: 'repository-name',
      description: 'Repository description',
      language: 'JavaScript',
      stargazers_count: 150
    }
  }
})
```

## Supabase Setup

### 1. Create Supabase Project

1. Sign up at [supabase.com](https://supabase.com)
2. Create a new project
3. Note your project URL and anon key

### 2. Install Supabase CLI

```bash
npm install -g supabase
supabase login
supabase link --project-ref your-project-id
```

### 3. Create Edge Function

```bash
supabase functions new summarize-repo
```

### 4. Add Function Code

Create `supabase/functions/summarize-repo/index.ts`:

```typescript
import { serve } from "https://deno.land/std@0.168.0/http/server.ts"

const GROQ_API_KEY = Deno.env.get('GROQ_API_KEY')

serve(async (req) => {
  const headers = {
    'Access-Control-Allow-Origin': '*',
    'Access-Control-Allow-Headers': 'authorization, x-client-info, apikey, content-type',
    'Content-Type': 'application/json',
  }

  if (req.method === 'OPTIONS') {
    return new Response('ok', { headers })
  }

  try {
    const { type, data } = await req.json()

    const prompt = type === 'profile'
      ? `Summarize this GitHub profile in 3-4 sentences: ${JSON.stringify(data)}`
      : `Summarize this repository in 2-3 sentences: ${JSON.stringify(data)}`

    const response = await fetch('https://api.groq.com/openai/v1/chat/completions', {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${GROQ_API_KEY}`,
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({
        model: 'mixtral-8x7b-32768',
        messages: [
          { role: 'system', content: 'You are a technical analyst.' },
          { role: 'user', content: prompt }
        ],
        max_tokens: 200,
      }),
    })

    const result = await response.json()
    const summary = result.choices[0]?.message?.content

    return new Response(JSON.stringify({ summary }), { headers })
  } catch (error) {
    return new Response(JSON.stringify({ error: error.message }), { headers, status: 500 })
  }
})
```

### 5. Set Secrets and Deploy

```bash
supabase secrets set GROQ_API_KEY=your_groq_api_key
supabase functions deploy summarize-repo
```
